These are the applications that are needed to be installed to run this system.

1. [`node` (via `nvm`)](https://github.com/nvm-sh/nvm#installing-and-updating)
2. [`python3`](https://www.python.org/downloads/)
3. [`virtualenv`](https://virtualenv.pypa.io/en/latest/)
4. [`jq`](https://stedolan.github.io/jq/)
5. [`gcloud`](https://cloud.google.com/sdk/install)
6. [`aws-cli`](https://docs.aws.amazon.com/cli/latest/userguide/cli-chap-install.html)
